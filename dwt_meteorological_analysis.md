## 1. 원본 이미지 (Original, 512×512)

한반도 전역의 **지표면 일사량(GHI, Global Horizontal Irradiance)** 분포입니다.

- **밝은 주황~노란 영역**: 맑은 하늘 → 높은 일사량 (직달일사 + 산란일사 모두 도달)
- **어두운 보라~갈색 영역**: 구름 차폐 → 일사량 급감 (주로 적운·층적운에 의한 차단)
- **복잡한 패턴**: 여름철 한반도는 해양성 기단과 대륙성 기단의 경계에서 불규칙한 적운 발달이 활발하여 구름–맑음의 경계가 매우 불규칙적

## 2. Level 1 분해 (256×256) — 대규모 기상 구조

### LL (Low-Low, 근사 성분) — 대기 순환 패턴
| 의미 | 설명 |
|------|------|
| 기상학적 | **종관 규모(Synoptic scale)의 일사량 분포**. 고기압/저기압 시스템에 의한 광역 구름 분포를 반영 |
| 특징 | 원본과 거의 동일한 형태 — **전체 에너지의 97%**를 담고 있음 |
| 물리적 | 수십~수백 km 규모의 일사 변동. 기압 배치, 전선대 위치, 해륙풍 수렴대 등이 지배 |

### HL (Horizontal Edge) — 동서 방향 구름 경계
| 의미 | 설명 |
|------|------|
| 기상학적 | **수평(동서) 방향의 급격한 일사 변화** = 남북으로 뻗은 구름 전선(cloud front) |
| 해석 | 밝은 점: 맑음→구름 또는 구름→맑음의 수평 전이 지점. 주로 **해안선을 따라 발달하는 해무(sea fog) 경계** 또는 **전선형 구름대의 남북 경계** |
| 왜 약한가 | 여름철 한반도는 남서풍 지배 → 구름 이동이 주로 SW→NE 방향이라 순수한 동서 경계는 상대적으로 적음 |

### LH (Vertical Edge) — 남북 방향 구름 경계
| 의미 | 설명 |
|------|------|
| 기상학적 | **수직(남북) 방향의 급격한 일사 변화** = 동서로 뻗은 구름 경계 |
| 해석 | **장마전선의 위치**가 이 성분에 강하게 반영됨. 7월 1일은 장마 시즌으로, 동서 방향으로 길게 놓인 정체전선(Mei-yu front)이 남북 방향의 급격한 일사 차이를 유발 |
| 특징 | HL보다 에너지가 약간 더 큼 — 장마전선의 동서 연장선이 남북 그래디언트를 생성 |

### HH (Diagonal Edge) — 대각선 방향 미세 텍스처
| 의미 | 설명 |
|------|------|
| 기상학적 | **대각선 방향의 급격한 변화** = 적운(Cumulus) 군집의 불규칙 경계 |
| 해석 | 단일 적운 또는 적란운의 **테두리(cloud edge)**에서 발생하는 일사 급변. 구름 그림자(cloud shadow)의 비정형 윤곽 |
| 왜 중요한가 | 태양광 발전 예측에서 가장 어려운 성분 — 개별 구름의 이동과 발달을 나타내며, 수분~시간 단위의 **램프 이벤트(ramp event)**의 원인 |

## 3. Level 2 분해 (128×128) — 중규모 기상 구조

### LL → LL (근사의 근사) — 광역 기압계
- **64×64 수준의 일사 분포** ≈ 한반도를 ~16km 격자로 본 것
- 고기압 중심 아래의 맑은 영역 vs 저기압/전선대 아래의 흐린 영역
- 수치예보모델(NWP)의 격자 해상도와 비슷한 수준의 정보

### LL → HL/LH/HH (중규모 디테일)
| 서브밴드 | 기상 현상 |
|----------|-----------|
| HL | **메소스케일 대류 시스템(MCS)**의 수평 경계. 중규모 전선의 위치 |
| LH | **산악 효과에 의한 구름 발달**. 태백산맥 풍하측의 맑음/흐림 경계 |
| HH | **대류 셀(convective cell)**의 조직화 패턴. 열적 대류에 의한 구름 클러스터링 |

### HF paths (ch 4-15) — Level 1에서 분기한 고주파 경로
맨 아래 행의 4개 이미지는 Level 1의 고주파 성분(HL, LH, HH)을 한 번 더 분해한 것입니다.

| 채널 | 원본 경로 | 기상학적 의미 |
|------|-----------|--------------|
| ch4 (HL→LL) | 수평 에지의 저주파 | **전선형 구름대의 대규모 윤곽** — 전선의 전체적인 위치와 방향 |
| ch8 (LH→LL) | 수직 에지의 저주파 | **산악 구름의 대규모 분포** — 지형에 의한 체계적 구름 패턴 |
| ch12 (HH→LL) | 대각 에지의 저주파 | **대규모 대류 조직 패턴** — 구름 클러스터의 공간적 배열 |
| ch15 (HH→HH) | 대각 에지의 고주파 | **개별 구름의 미세 텍스처** — 구름 내부의 두께 변동 (가장 노이즈에 가까움) |

## 4. Level 3 분해 (64×64) — 최대 규모 기상 구조

### LL→LL→LL (Coarsest, 64×64) — "기상 지도"
- **한반도 전체의 일사 편향(bias)**을 담은 가장 거친 표현
- 이 64×64 하나가 전체 에너지의 대부분을 차지
- 기상학적으로는 **일기도(synoptic chart)** 수준의 맑음/흐림 분포
- **모델이 가장 먼저 학습해야 하는 핵심 성분**

### LL→LL→HL/LH/HH (64×64 디테일)
- 64×64에서의 디테일 → 한반도를 ~32km 격자 8×8 블록으로 나눈 수준
- **기압계 경계, 광역 전선대의 대략적 위치** 정보
- 블록아티팩트처럼 보이는 것은 64×64 해상도에서의 양자화 효과

## 5. 에너지 분포와 CP6 문제의 기상학적 근본 원인

```
LL chain (ch 0-3)     = 88.1%  ← 대기 순환 패턴
LL→HL/LH/HH (ch 4-15) =  5.1%  ← 구름 경계/전선
HF paths (ch 4-15)    =  6.8%  ← 미세 구름 텍스처
Level 1 LL alone       = 97.0%
```

### 왜 에너지가 LL에 집중되는가?

> [!IMPORTANT]
> 일사량 데이터의 물리적 특성상, **맑은 하늘 영역이 공간적으로 지배적**이며 그 값이 구름 영역보다 수배 크기 때문에 자연스럽게 LL(저주파)에 에너지가 편중됩니다.

기상학적으로 이는 다음을 의미합니다:

1. **맑은 하늘 = DC 성분**: 구름이 없는 영역의 일사량은 태양 고도와 대기 투과율에 의해 결정되며, 공간적으로 매우 균일(slowly varying). 이것이 LL의 88%를 구성합니다.

2. **구름 = AC 성분**: 구름에 의한 일사 변동은 전체 신호 대비 "교란(perturbation)"에 해당. 에너지 비중은 작지만 **예측 정확도의 핵심**입니다.

3. **CP6 문제의 본질**: 모델이 88%의 에너지를 차지하는 "맑은 하늘 배경"을 복원하는 데 대부분의 gradient를 소비하고, 정작 중요한 **구름 경계(5~7%)를 학습하지 못함** → SSIM 정체(~0.37)

### 이것이 실험 설계에 주는 시사점

| 실험 | 타겟 서브밴드 | 기상학적 목표 |
|------|-------------|--------------|
| **H/I** (L1/Charbonnier) | 모든 주파수 균등 | 맑은 하늘 복원에 gradient 매몰 → 실패 |
| **J** (Focal Freq. Loss) | 복원 어려운 주파수 자동 증폭 | **구름 경계에 자동 집중** — HL/LH/HH 성분 학습 촉진 |
| **M** (Variance Loss) | 채널별 σ 기반 가중 | **ch0(LL) gradient를 억제**하고 ch1-15를 증폭 → 구름 디테일 학습 유도 |

> [!TIP]
> **J와 M이 성공한다면**, 그것은 모델이 "맑은 하늘(88%)"을 적당히 학습한 상태에서 "구름 경계(12%)"에 집중할 수 있게 된다는 의미입니다. 이것이 SSIM 0.37 → 0.50+ 돌파의 핵심 메커니즘입니다.
