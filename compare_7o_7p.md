# 7O vs 7P 비교 분석 리포트

## 실험 개요

| 실험 | 방법 | 변경점 |
|:---:|------|------|
| **BL** | Baseline (CosineLR + Min-SNR) | — |
| **7O** | Per-Channel Min-SNR | **학습 시** 채널별 SNR 가중치 적용 |
| **7P** | Baseline + BayesShrink | **추론 시** 고주파 대역 Wavelet 디노이징 |

> [!IMPORTANT]
> 7O는 **학습(Train) 단계 개입**, 7P는 **추론(Inference) 후처리**. 두 방법은 직교하므로 시너지(7Q)가 기대됩니다.

## 핵심 트레이드오프 해석

### 🟢 7O의 강점: Pixel Accuracy (MSE, PSNR, Latent_Low)

Per-Channel Min-SNR은 **LL(저주파) 채널의 gradient를 억제**하고 **HH(고주파)의 gradient를 증폭**합니다.

```
ch0 (LL, σ²=13.82): SNR > γ → gradient 억제 → 과적합 방지
ch63(HH, σ²=0.004): SNR < γ → 풀 gradient → 고주파 학습 촉진
```

결과:
- **Latent_MSE_Low 0.154 → 0.151** (-2.3%): LL 과적합 억제 성공
- **MSE 0.044 → 0.044** (-1.6%): 전체 pixel 오차 감소
- **PSNR +0.24dB**: 미세하지만 일관된 개선

### 🔵 7P의 강점: Perceptual Quality (SSIM, LPIPS, Lat_High)

BayesShrink는 **추론 후 고주파 대역의 Wavelet 계수를 soft-thresholding**합니다.

결과:
- **Latent_MSE_High 0.0079 → 0.0063** (-20%): 고주파 노이즈 대폭 제거
- **SSIM 0.500 → 0.483** (-3.4%): SSIM 하락이 7O(-10%)보다 훨씬 적음
- **LPIPS 0.612 → 0.664** (+8.5%): perceptual 품질 악화가 7O와 거의 동일

### ❌ 공통 약점: FID 대폭 악화

| | BL | 7O | 7P |
|---|:---:|:---:|:---:|
| FID | **193.5** | 252.4 | 252.1 |

두 방법 모두 FID가 ~58포인트 악화. 이는 **분포적 다양성 저하**(패턴이 더 평균적으로 수렴)를 의미합니다.

## 7Q(시너지) 기대치

| 메트릭 | 7O 기여 | 7P 기여 | 예상 효과 |
|--------|---------|---------|----------|
| MSE | ✅ 학습 시 채널 밸런싱 | ❌ 후처리 MSE 악화 | **7O가 MSE 잡고, 7P가 perceptual 보전** |
| SSIM | ❌ -10% 하락 | ✅ -3%만 하락 | BayesShrink가 7O의 SSIM 하락 완충 |
| FID | ❌ +30% 악화 | ❌ +30% 악화 | ⚠️ 시너지 불확실, 추가 악화 가능 |
| Lat_High | ❌ 미미 | ✅ -20% 감소 | 7P의 고주파 정제가 작동 |

> **핵심 기대**: 7O의 MSE/PSNR 이점 + 7P의 SSIM/Lat_High 이점을 동시에 취하되, **FID 악화가 누적되지 않는지**가 7Q의 성패를 결정합니다.
