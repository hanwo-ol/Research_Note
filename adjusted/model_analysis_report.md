# ST-DiT P19 마스터 모델 분석 보고서

본 문서는 P19 단계에서 최종적으로 확보된 마스터 모델 (`Stage8_VPred_3yr_SGP0.01_FFL_BS16`)의 내부 동작 및 물리적 건전성을 검증하기 위해 작성된 심층 분석 보고서입니다. 3년치(2021-2023) 테스트 세트 전체 데이터를 사용하여 엄밀한 평가를 진행하였습니다.

## 1. 모델 가중치 및 구조적 건전성 분석 (Static Weight Health)
✅ **분석 결과: 모델 붕괴 위험 없음 및 각 모듈의 정상 작동 확인**

*   **파라미터 상태**: 모델의 총 가중치 개수는 약 5,225만 개이며, 결측치(NaN)는 0개입니다. 가중치의 분포는 최솟값 -3.00, 최댓값 3.08로 매우 안정적인 범위를 유지하고 있습니다. 극단적으로 0에 가까운 희소성(Sparsity) 비율은 4.25% 수준으로 적절한 활성화를 보입니다.
*   **잔차 연결 및 게이트(Gate) 동작**:
    *   **물리적 시간 적응 게이트 (Timestep Gate)**: 평균 편향(Bias) 값이 2.01로, 약 88.2%의 비율로 열려 있습니다. 이는 모델이 물리적 시간 정보를 대부분 수용하면서도 필요한 경우 일부 정보를 선별적으로 차단하고 있음을 의미합니다.
    *   **심층 스킵 연결 (Deep Connection Gate)**: 평균 편향 값 0.95로, 약 72.2%의 개방률을 기록했습니다. V-Prediction 환경에서 필수적인 입력-출력 간의 스킵 연결이 원활하게 작동하고 있습니다.

## 2. 어텐션 맵 정보량 분포 (Attention Entropy)
✅ **분석 결과: 정보의 고른 분산 및 국소 함몰(Mode Collapse) 징후 없음**

*   전체 테스트 데이터셋을 대상으로 1회 DDIM 추론을 수행하여 주의 집중(Attention) 맵의 섀넌 엔트로피(Shannon Entropy)를 측정하였습니다. 총 32,632개의 어텐션 헤드를 측정한 결과 평균 엔트로피는 **7.1299 bits**로 산출되었습니다.
*   본 모델의 시간-공간 컨텍스트 길이가 최대 320 토큰(5프레임 × 64토큰)임을 고려할 때, 이론상 최대 엔트로피는 약 8.32 bits입니다. 측정된 7.13 bits는 모델이 한 두 개의 특정 토큰에만 극단적으로 과적합하여 함몰되지 않고, 제공된 다중 프레임 정보를 비교적 고르게 참조하여 시공간 맥락을 효과적으로 활용하고 있음을 증명합니다.

## 3. 주야간 물리적 일관성 (Day/Night Physics)
- **목적**: 주야간 전이에 따른 도메인 변위 시 오차 증폭 여부 평가
- **분석 스크립트**: `analyze_domain.py`
- **조건**: 전체 테스트셋 6,336 샘플 (Effective 4,079 샘플), DDIM 10 스텝, Guidance Scale 1.0 (CFG off)
- **주간 (Daytime, 4,079 샘플) 오차**:
  - MSE: 0.0348
  - MAE: 0.1065
  - SSIM: 0.5581
- **야간 (Nighttime, 0 샘플) 오차**:
  - 3개년 데이터셋 구성 시 야간 데이터가 물리적 필터링을 통해 완전히 제거되었음을 검증.
- **결론**: 야간 노이즈 유입이 원천 차단되었으며 주간 예측 시 0.0348 MSE의 매우 우수한 예측 성능을 유지.

## 4. 주파수 대역별 도메인 오차 (Frequency Error)
- **목적**: 예측된 이미지의 저주파(형태/조도)와 고주파(디테일/질감) 대역별 재구성 오차 편차 확인
- **분석 스크립트**: `analyze_domain.py`
- **저주파 (LL) 대역 평균 MSE**: 0.1202
- **고주파 (HL, LH, HH) 대역 평균 MSE**: 0.0063
- **결론**: Focal Frequency Loss와 Deep Stem의 효과로 고주파수(질감 및 급격한 구름의 엣지) 대역의 오차가 극히 낮음 (0.0063). 모델의 잔여 오차(Residual Error)는 주로 저주파 대역(0.1202)의 거시적인 조도 변화 및 전역적 구름의 이동 궤적에서 발생함을 증명. 
*   이러한 수치는 P19 단계에서 새롭게 도입된 주파수 집중 손실(Focal Frequency Loss) 기법이 높은 주파수 대역에서의 노이즈 및 흐림(Blur) 현상을 억제하는 데 강력한 효과를 발휘하고 있음을 뒷받침합니다.

## 종합 결론
P19 마스터 모델은 기존 EMA 기법에서 발생했던 모델 붕괴 현상을 완전히 극복하였으며, 가중치의 건전성, 어텐션의 분산 구조, 주파수 대역별 오차 등 모든 지표에서 안정적이고 신뢰할 수 있는 상태임이 확인되었습니다. 따라서 본 모델 구조(V-Prediction + SGP + FFL)를 향후 이어질 개선 연구의 새로운 기준선(Baseline)으로 채택하는 것이 타당합니다.
